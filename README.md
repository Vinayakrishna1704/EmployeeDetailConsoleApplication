##  Overview
This project is a **C# console application** built using **.NET 8**, designed to demonstrate the use of **LINQ (Language Integrated Query)** for data manipulation and analysis.  
It simulates an **Employee Management System**, allowing users to interact with employee data through a simple menu-driven interface.

The primary objective of this project is to **showcase how LINQ queries simplify data filtering, sorting, grouping, and aggregation** in real-world scenarios.

---
